A sleek, terminal-based productivity suite featuring a Pomodoro timer and a task manager, built in Java.

## Features
- **Task Manager**: Add, list, and complete tasks directly from the terminal.
- **Pomodoro Timer**: (Coming soon) A focused work-break timer to boost productivity.

## How to Run
Compile the code:
```bash
javac src/*.java
```

Run the application:
```bash
java -cp src Main
```
